uroboros is A GNU/Linux monitoring and profiling tool focused on single processes.

**WORK IN PROGRESS, ALPHA STAGE, EXPECT CRASHES AND WEIRD STUFF**

## Usage

    make uro

To monitor by pid:

    ./_build/uro -pid 1234

To search by process name:

    ./_build/uro -name test-process

For more options:
    
    ./_build/uro -help
